Images:  Images can be inserted into web pages using 'img' tag . it will take attributes such as
      src  -- used for source path of Images
      alt -- used as an alternative text that works when source is not loaded.
img is a self closing tag. A self closing tag will not have specific closing tag. These tags are also called unpaired tags
HTML is not case sensitive.

There are different types of images supported by HTML/Browsers such as
     JPG/JPEG  -- normal images
     PNG  -- Transparent background images
     GIF  -- animated Images
     SVG  -- scalable vector graphics
     webp -- High Quality Images

Modern HTML Supports Responsive Images such as displaying one image for desktop and another image for mobile etc.., it is possible using 'picture' tag
   for picture tag we take source using source tag with 'srcset' attribute and 'media' attribute to specify the device width
   Always sizes should be decreasing order and always default image should be at end.
